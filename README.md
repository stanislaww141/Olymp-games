# Olympic Games History Dashboard (1896-2016)

Интерактивный аналитический отчет, разработанный в Power BI.

<img width="100%" alt="Обложка дашборда" src="https://github.com/user-attachments/assets/0c45ba26-a80d-4c78-8691-35536b5fd2fa" />

## Описание проекта

Цель проекта - визуализировать исторические данные Олимпийских игр за 120 лет. Дашборд позволяет анализировать результаты стран, демографию атлетов и динамику медального зачета, а также предоставляет удобный поиск информации по конкретным спортсменам.

**Источник данных:** [Kaggle: 120 years of Olympic history: athletes and results](https://www.kaggle.com/datasets/heesoo37/120-years-of-olympic-history-athletes-and-results).

## Технический стек

* **Инструмент:** Microsoft Power BI Desktop
* **Языки:** DAX, HTML/CSS
* **Дизайн:** Figma
